Tuy nhiên có 1 số lưu ý sau Đối với việc setup token cho auto_deploy name: Deploy to GH Pages
on: [push]
jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    steps:
    - name: Checkout
      uses: actions/[email protected]

    - name: Build and Deploy
      uses: jenkey2011/vuepress-[email protected]
      env:
        ACCESS_TOKEN: ${{ secrets.ACCESS_TOKEN }}
        TARGET_REPO: hoangpn256/hoangpn.github.io
        TARGET_BRANCH: gh_pages
        BUILD_SCRIPT: yarn && yarn build
        BUILD_DIR: src/.vuepress/dist/
 Để ý rằng có biến secrets.ACCESS_TOKEN -> đây là access token của tài khoản github của bạn. Để lấy được nó hãy làm theo hướng dẫn sau đây Đối với việc chia thư mục viết bài Các bạn có thể tham khảo cấu trúc và config ở repo của mình.
